# Extending Flare

Extending Flare is easy! Create your own list and filter element types by tagging any of your services with one
of our provided PHP 8 attributes, or hook into the existing list and filter element types with
Callbacks and Events. Stick any of your favorite or custom Symfony FormTypes onto a filter element and have your
users filter your lists with ease.

## Overview

Flare provides a set of predefined list and filter element types that can be used out of the box. However, you can
easily extend Flare with your own custom list and filter element types. This allows you to tailor the filtering and listing experience to your specific needs.

In this documentation, we will cover how to create custom list and filter element types, as well as how to hook into existing ones.
